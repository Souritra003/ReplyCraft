## Overview  
ReplyCraft is a browser extension that integrates seamlessly with Gmail to provide context-aware, AI-generated replies. Built with **Spring Boot** (REST API backend) and **React.js** (frontend), it leverages the **Gemini API** to craft intelligent responses, saving time and boosting productivity for everyday email communication.
